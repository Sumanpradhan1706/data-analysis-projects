# 01 — Zomato Data Analysis

Project summary
- A data exploration and visualization project analyzing Zomato restaurant data to extract insights about ratings, cuisines, location trends, and pricing.

Notebooks
- `anaconda_projects/db/01_Zomato_Data_Analysis.ipynb` — primary analysis and visualizations.

Data
- `anaconda_projects/db/Zomato-data-.csv` — original dataset used for analysis. Keep this file in the `dataset/` folder if you restructure.

Key analyses
- Data cleaning and normalization
- Descriptive statistics by city and cuisine
- Visualizations: rating distributions, price vs. rating, top cuisines

Dependencies
- Python 3.8+; recommended packages: `pandas`, `numpy`, `matplotlib`, `seaborn`, `plotly`, `jupyter`.

Quick start
1. Create environment: `conda create -n zomato_analysis python=3.8 -y`
2. Activate: `conda activate zomato_analysis`
3. Install: `pip install pandas numpy matplotlib seaborn plotly jupyter`
4. Launch: `jupyter notebook anaconda_projects/db/01_Zomato_Data_Analysis.ipynb`
