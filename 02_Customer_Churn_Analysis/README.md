# 02 — Customer Churn Analysis

Project summary
- Analysis and predictive modeling of customer churn using the Telco Customer Churn dataset. The project demonstrates EDA, feature engineering, and baseline classification models.

Notebooks
- `anaconda_projects/db/Customer_Churn_Analysis.ipynb` — EDA, preprocessing, modeling, and evaluation.

Data
- `anaconda_projects/db/Telco-Customer-Churn.csv` — source dataset used for modeling.

Key analyses
- Exploratory data analysis and visualizations
- Feature engineering and handling categorical variables
- Baseline models: Logistic Regression, Random Forest
- Model evaluation and interpretation

Dependencies
- Python 3.8+; recommended: `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`, `xgboost` (optional).

Quick start
1. `conda create -n churn_analysis python=3.8 -y`
2. `conda activate churn_analysis`
3. `pip install pandas numpy scikit-learn matplotlib seaborn xgboost`
4. `jupyter notebook anaconda_projects/db/Customer_Churn_Analysis.ipynb`
